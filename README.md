# 📰 Premium Portal Lokalny - w stylu xyz.pl

Elegancki portal lokalnych newsów ekonomicznych inspirowany designem xyz.pl. Nowoczesny, minimalistyczny i skupiony na jakości treści.

## 🎯 **Nowa transformacja - styl xyz.pl**

Portal został przeprojektowany na wzór **xyz.pl** - premium medium ekonomicznego:

### ✨ **Nowe cechy designu:**
- **Elegancka typografia** - Inter + Crimson Text jak w xyz.pl
- **Minimalistyczny layout** - fokus na treści, nie na ilości
- **Premium feel** - subskrypcje, exclusive content
- **Pogłębione analizy** zamiast szybkich newsów
- **Wyrafinowane kolory** - monochromatyczna paleta
- **Business-oriented** - ekonomia i biznes lokalny

## 🎨 **Design System**

### **Colors:**
```css
--primary-color: #1a1a1a     /* Deep black */
--accent-color: #007acc      /* Professional blue */
--text-secondary: #666666    /* Subtle gray */
--background-light: #fafafa  /* Clean background */
```

### **Typography:**
- **Headlines**: Crimson Text (serif)
- **Body**: Inter (sans-serif)
- **Hierarchy**: Clear, readable, elegant

### **Layout:**
- **Grid**: 2/3 main + 1/3 sidebar
- **Spacing**: Generous whitespace
- **Cards**: Subtle shadows, hover effects

## 🚀 **Development Workflow**

### **Automatic Deployment:**
```
Code change → GitHub → Actions → Firebase → Live!
```

### **Development Setup:**
1. Clone repository
2. Install Firebase CLI: `npm install -g firebase-tools`
3. Login: `firebase login`
4. Serve locally: `firebase serve`
5. Deploy: `firebase deploy`
